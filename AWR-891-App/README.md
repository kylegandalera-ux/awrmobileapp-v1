# Adventist World Radio 89.1 FM App

Open `index.html` in a browser to view the mobile app mockup.

Editable links live near the top of `app.js`:

- `appConfig.donationUrl`
- `appConfig.stations[].youtubeUrl`
- `appConfig.bibleAudio[].url`
- `appConfig.moreLinks[].url`

The in-app `Edit links` button also lets you test a station YouTube URL and donation URL without editing code.
